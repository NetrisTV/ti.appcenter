MSEnable

@protocol MSEnable <NSObject>

Protocol to define an instance that can be enabled/disabled.

  • -setEnabled:andDeleteDataOnDisabled:

    Enable/disable this instance and delete data on disabled state.

    Declaration

    Objective-C

    - (void)setEnabled:(BOOL)isEnabled andDeleteDataOnDisabled:(BOOL)deleteData;

    Swift

    func setEnabled(_ isEnabled: Bool, andDeleteDataOnDisabled deleteData: Bool)

    Parameters

    isEnabled

    A boolean value set to YES to enable the instance or NO to disable it.
    deleteData

    A boolean value set to YES to delete data or NO to keep it.
